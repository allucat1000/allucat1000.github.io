Useful Debian Linux Commands

Here are some Linux commands for Debian, so that you don't have to remember so much stuff.

• Install scripts

Gnome install

sudo apt update

sudo apt install gnome

sudo reboot

XFCE install

sudo apt update

sudo apt install xfce4 xfce4-goodies

sudo apt install lightdm

sudo reboot

KDE Install

sudo apt update

sudo apt install kde-full

sudo reboot

• Miscellaneous

Flatpak install (terminal)

sudo apt update

sudo apt install flatpak

sudo flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

reboot

Give sudo perms

usermod -aG sudo (replace with the account to give sudo to)

reboot
